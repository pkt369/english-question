English Question Practice is a free static web app for Korean English learners. 400 questions across 4 levels (Beginner/Intermediate/Advanced/Expert). No backend, no accounts. Deployment: Vercel/Netlify/GitHub Pages.

**Why:** Fill the gap between expensive AI conversation apps ($8-15/mo) and unstructured blog question lists. Target Korean study group culture.

**How to apply:** All PDCA phases should reference the PRD at `docs/00-pm/english-question.prd.md`. Beachhead segment is Korean English study group leaders/members. Key risk: content quality and retention.
